# Parachute
A stand still shoot them down game

### Instructions:
* Use the left and right arrow keys to rotate turret
* Press the spacebar or right-click on the touchpad to shoot
* You have 3 lives, shoot down parachuters
* Press Q to quit
* Press R to restart
